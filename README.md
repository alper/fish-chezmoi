# fish-chezmoi

[Chezmoi](https://github.com/twpayne/chezmoi) abbreviations

## 🚀 Installation

Using [fisher](https://github.com/jorgebucaran/fisher)

```bash
fisher add gazorby/fish-chezmoi
```

## 🔧 Usage

| Abbreviation | Command        |
| ------------ | -------------- |
| cm           | chezmoi        |
| cma          | chezmoi add    |
| cmap         | chezmoi apply  |
| cmu          | chezmoi update |
